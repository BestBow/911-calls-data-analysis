# 911 Calls Data Analysis

This project explores emergency 911 call data to identify patterns in emergency services usage.

## Dataset
911 call dataset containing information about:
- call type
- location
- timestamp
- township

## Analysis Performed
- Data cleaning
- Feature engineering
- Time based analysis
- Visualization of call patterns

## Technologies
- Python
- Pandas
- Seaborn
- Matplotlib
- Plotly

## Key Insights
- EMS calls were the most frequent emergency category.
- Call frequency varies by hour and day of the week.
- Certain townships report higher emergency activity.
